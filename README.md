🌤 Weather App

A simple and responsive Weather Web Application that allows users to check real-time weather information for their current location or search weather conditions for any city around the world.

🔗 Live Demo: https://kylerose-weatherapp.netlify.app/

📌 Features

📍 Current Location Weather

Grant location access to instantly view weather data for your current location.

🔎 Search by City

Search for weather conditions in any city worldwide.

🌬 Weather Details

Displays important weather metrics such as:

Temperature

Conditions

⚡ Fast & Responsive

Designed for quick weather checks on both desktop and mobile devices.
